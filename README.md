## Fitur Utama
- **Fitur 1**: Lebih menarik dibanding tampilan default awal termux
- **Fitur 2**: enak dilihat

## Instalasi

1. Instal dependensi
   ```python
   pkg update && pkg upgrade
   ```
 2. Instal dependensi
   ```python
   pkg install python
   ```
2. Instal dependensi
   ```python
   pkg install git
   ```
1. Clone repositori ini
   ```github

   ```
2. Instal dependensi
   ```python
   cd themezdz
   ```
2. Instal dependensi
   ```python
   python termux_theme.py
   ```
2. Instal dependensi
   ```python
   source ~/.bashrc
   ```
